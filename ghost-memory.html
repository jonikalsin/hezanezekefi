The Ghost Memory (Japanese: ゴーストメモリ Ghost Memory) is a type of held item introduced in Generation VII. It is one of the memories that are associated with Silvally. It corresponds to the Ghost type.

In the core series games

Price

Effect

In battle

If the holder uses Multi-Attack, it becomes a Ghost-type move.

RKS System changes a Silvally holding this item to its Ghost-type form. Item-manipulating effects (such as Trick and Fling) cannot give Silvally a memory or remove a memory from Silvally.

Outside of battle

RKS System changes a Silvally holding this item to its Ghost-type form.
